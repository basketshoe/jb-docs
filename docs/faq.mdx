## Cos'è il jailbreak?

"Jailbreak" è un termine utilizzato per descrivere il processo che permette di sbloccare le vere potenzialità dei dispositivi iOS/iPadOS. Il sistema operativo di Apple è noto per la sua sicurezza e questo è dovuto in parte alle sue restrizioni. Il jailbreak rimuove queste restrizioni e permette la piena customizzazione del device.

## Il jailbreak è legale?

Il jailbreak è completamente legale secondo la direttiva UE 2009/24/EC. Gli utenti hanno il diritto di modificare le funzionalità software di un dispositivo senza violare le leggi sul copyright.

Inoltre, non invalida la garanzia.

# Cos'è un Tweak?

È un pacchetto software installabile che permette di effettuare modifiche sul look o sulle funzionalità del dispositivo.

Di solito non hanno un'icona nella home, ma un'opzione apposita nell'app Impostazioni per cambiare qualche preferenza.

## Cos'è un Package Manager?

Sono software che consentono l'installazione di software per la customizzazione dell'iDevice. Sileo è un esempio di package manager.

Possono essere visti come l'App Store, dove puoi installare software a pagamento o gratuito.

## Cos'è una Repo?

È di fatto un database che contiene diversi tweak installabili. Quando viene aggiunta una repo al package manager, è possibile vedere i tweak che contiene ed installarli (mantenendoli aggiornati).

## Cos'è il "respring"?

Consiste nel riavvio della home e dell'interfaccia utente. La maggior parte delle volte è possibile installare un tweak e applicare le modifiche tramite respring, senza dover eseguire un vero riavvio.

Il respring non disattiva il jailbreak, a differenza del riavvio.

## Cos'è la Recovery Mode?

È una modalità di protezione del device che permette di ripristinare il dispositivo tramite un'installazione da zero di iOS/iPadOS.

## Cos'è la DFU Mode?

È una modalità per interagire col dispositivo senza caricare il sistema operativo o il boot loader. È diversa dalla Recovery Mode ed è considerata l'ultima spiaggia in caso le cose si mettano male.

## Cosa posso fare col jailbreak?

* Ottenere il pieno controllo sul dispositivo
* Installare temi per cambiare il look del device
* Installare tweak per modificare il funzionamento di iOS/iPadOS

## Cosa devo sapere prima di iniziare?

* È **fortemente** consigliato eseguire un backup prima di iniziare
* Tieni il dispositivo in carica mentre applichi il jailbreak
* Leggi la guida di tuo interessa **da cima a fondo**
* Potresti essere vulnerabile a più virus o malware una volta jailbroken
* Se non sai cosa stai facendo, non farlo

## Il jailbreak è morto?

La speranza è l'ultima a morire.
