## Premessa
È ora disponibile [Dopamine 2](https://ellekit.space/dopamine/) che consente il jailbreak di iOS 15.0 - 16.6.1 su dispositivi con processori da A9 a A16 e M1 - M2.

È riportata qui sotto la tabella di compatibilità (fonte: [Discord di r/jailbreak](https://discord.gg/jb))
Se non conosci il tuo processore digita ```!processori``` nel gruppo o visita [questo sito](https://appledb.dev/device-selection/).

## Disinstalla Jailbreak Precedenti

:::info
Puoi saltare questa sezione se non disponi di un jailbreak
:::

È **molto importante** rimuovere roothide Bootstrap / Serotonin prima di installare Dopamine.
1. Rimuovi l'injection di tutte le app nell'app Bootstrap
2. Premi su "Select All" e "Clean" nel tab varClean dell'app roothide Manager (```com.roothide.manager```)
3. Rimuovi ```/var/mobile/Serotonin.jp2``` (o ```/rootfs/var/mobile/Serotonin.jp2```)
4. Riavvia il dispositivo
5. Premi "Uninstall" nell'app roothide Bootstrap

:::danger
Se non rimuovi l'injection delle app, queste non si avvieranno con Dopamine (incluse quelle di sistema).
:::

## Installazione

Installare l'app sul dispositivo è possibile tramite Sideloading (digita ```!altstore``` o ```!sideloadly``` sul gruppo) o tramite TrollStore (digita ```!trollstore``` sul gruppo).

:::warning
È ALTAMENTE consigliata l'installazione tramite [Trollstore](/docs/trollstore)!
:::

### Link
Collegati a [questo link](https://github.com/opa334/Dopamine/releases/latest), se vuoi installare Dopamine tramite sideloading scarica il file ```Dopamine.ipa``` altrimenti, se vuoi installarla tramite TrollStore, scarica ```Dopamine.tipa```.

*Nota che per le versioni beta di iOS è necessario installare Dopamine tramite TrollStore*

Una volta installata l'app:
1. Riavvia il dispositivo
2. Apri l'app di Dopamine immediatamente dopo il riavvio
3. Tocca su "Jailbreak!"

:::info
Se l'app crasha o il device si riavvia inaspettatamente e il jailbreak non è installato, riavvia e riesegui l'exploit finché non funziona.
:::

Sileo dovrebbe ora essere installato nella homescreen e potrai utilizzarlo per installare tweak o temi.

## Installare il software necessario
1. Apri Sileo
2. Tocca su "Sorgenti"
3. Aggiorna le Repo
4. Cerca il tweak ElleKit
5. Tocca "Ottieni"
6. Tocca la bara "Coda" in fondo alla pagina
7. Tocca su "Conferma"
8. Una volta finito tocca "Restart SpringBoard"

:::info
Dopo aver eseguito tutti gli step, riavvia e riesegui il jailbreak.
:::
